# Real Estate Investment Calculator

A professional-grade, two-phase real estate investment analysis tool. Built for property consultants and investors evaluating pre-selling condominium investments.

## Live Demo

Open `index.html` in any modern browser. No server required — runs entirely client-side.

## Features

**Two-Phase Investment Model**
- **Phase 1 — Pre-Selling / Construction:** Reservation fee, down payment spread, monthly amortization during construction period, built-in equity calculation
- **Phase 2 — Post-Turnover / Rental:** Balance financing with interest, monthly rental income projection, occupancy rate, association dues, property tax, net cash flow analysis

**Investment Analysis**
- Total ROI and annualized return (CAGR)
- Gross and net rental yield
- Capital appreciation projection with interactive chart
- Year-by-year cash flow visualization
- Complete exit analysis with selling costs (CGT + broker)

**Comparison Mode**
- Property investment vs. bank time deposits vs. equities/REITs
- Same capital, same holding period — side-by-side returns
- Growth comparison chart over full analysis horizon

**PDF Export**
- Clean print-optimized layout
- Investment summary with all key metrics
- Comparison table included
- Professional disclaimer

## Usage

1. Open `index.html` in a browser
2. Fill in property details and assumptions
3. Click **Calculate Investment Analysis**
4. Review results across Analysis, Comparison, and Export tabs
5. Export as PDF via the Export Summary tab

## Deployment

### GitHub Pages
1. Rename `investment-calculator.html` to `index.html`
2. Push to a GitHub repository
3. Enable GitHub Pages in repository settings (Source: main branch)
4. Access at `https://yourusername.github.io/repository-name/`

### Any Static Host
Upload the single HTML file to Netlify, Vercel, or any static hosting provider.
